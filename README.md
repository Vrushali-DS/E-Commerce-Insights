# E-Commerce-Insights
# Overview
The E-Commerce Insights Dashboard is an interactive Power BI project designed to analyze sales performance, product trends, regional sales distribution, and customer payment behavior. The dashboard helps businesses monitor key performance indicators (KPIs) and make data-driven decisions.

# Objectives
Track overall sales performance.
Analyze product-wise sales trends.
Monitor regional sales distribution.
Understand customer payment preferences.
Identify top-performing products.
Evaluate order volume and product demand

# Dashboard Components
1. Sales Trend Analysis
Tracks sales performance over different time periods, helping identify business growth patterns and seasonal trends.

2. Sales by Region
Compares revenue generated across:
East
North
South
West
3. Top Products by Sales
Highlights the highest revenue-generating products, enabling better inventory and product strategy decisions.

4. Regional Sales Distribution
Visualizes the contribution of each region to total sales using a treemap.

5. Payment Method Analysis
Analyzes customer payment preferences across:
Cash
UPI
Net Banking
Card
6. Interactive Filtering
Allows users to dynamically filter dashboard insights by region for deeper analysis.

#  Tools & Technologies Used

DAX (Data Analysis Expressions)
Data Modeling
Data Visualization
